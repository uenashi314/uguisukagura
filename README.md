# ウグイスカグラ修正パッチ

## 紙の上の魔法使い

バージョン1.6で、予約特典を閲覧できるようにするパッチです  
[ダウンロード(patch2.xp3)](/kamimaho/patch2.xp3)  

DL(Fanza)版に適応する場合は、以下のDLLも追加してください。  
[ダウンロード(krmovie.dll)](/kamimaho/krmovie.dll)

上記のファイルをダウンロードし、ゲームをダウンロードしたフォルダに配置してください。
